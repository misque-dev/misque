# Hijri

Convert between Gregorian and Hijri (Islamic) calendars with formatting support. The Hijri calendar is a lunar calendar used by Muslims worldwide to determine religious dates and events.

> **Note**: AH (Anno Hegirae) marks the Islamic calendar era, beginning with the Prophet Muhammad's migration from Makkah to Madinah in 622 CE. Actual Hijri dates may vary by 1-2 days depending on local moon sighting traditions.

## Quick Start

```typescript title="quick-start.ts"
import { toHijri, formatHijri } from '@misque/hijri';

const result = toHijri(new Date());
if (result.success) {
  console.log(result.data); // { year: 1446, month: 6, day: 15 }

  const formatted = formatHijri(result.data);
  console.log(formatted); // "15 Jumada al-Thani 1446"
}
```

## Conversion

All conversion functions return a `Result<T>` type for error handling.

### Gregorian to Hijri

```typescript title="gregorian-to-hijri.ts"
import { toHijri, gregorianToHijri } from '@misque/hijri';

// From Date object
const result = toHijri(new Date(2024, 0, 1));
if (result.success) {
  console.log(result.data); // { year: 1445, month: 6, day: 19 }
}

// From year, month, day components
const result2 = gregorianToHijri(2024, 1, 1);
if (result2.success) {
  console.log(result2.data); // { year: 1445, month: 6, day: 19 }
}
```

### Hijri to Gregorian

```typescript title="hijri-to-gregorian.ts"
import { toGregorian, hijriToGregorian } from '@misque/hijri';

// From HijriDate object
const result = toGregorian({ year: 1446, month: 1, day: 1 });
if (result.success) {
  console.log(result.data); // Date object
}

// From year, month, day components
const result2 = hijriToGregorian(1446, 1, 1);
if (result2.success) {
  console.log(result2.data); // { year: 2024, month: 7, day: 7 }
}
```

### Full Conversion Result

```typescript title="full-conversion.ts"
import { convert } from '@misque/hijri';

// Get both dates and Julian Day
const result = convert(new Date());
if (result.success) {
  console.log(result.data.hijri); // { year: 1446, month: 6, day: 15 }
  console.log(result.data.gregorian); // { year: 2024, month: 12, day: 17 }
  console.log(result.data.julianDay); // 2460661.5
}
```

## Formatting

```typescript title="formatting.ts"
import { formatHijri, formatHijriArabic } from '@misque/hijri';

const hijri = { year: 1446, month: 9, day: 1 };

// Default format (medium)
formatHijri(hijri);
// "1 Ramadan 1446"

// Arabic month names
formatHijri(hijri, { useArabic: true });
// "1 رمضان 1446"

// Short format
formatHijri(hijri, { style: 'short' });
// "1/9/1446"

// Long format with AH suffix
formatHijri(hijri, { style: 'long' });
// "1 Ramadan 1446 AH"

// Full format with day of week
formatHijri(hijri, { style: 'full', includeDayOfWeek: true });
// "Friday, 1 Ramadan 1446 AH"

// Quick Arabic formatting
formatHijriArabic(hijri);
// "1 رمضان 1446 AH"
```

## Utilities

```typescript title="utilities.ts"
import {
  isHijriLeapYear,
  getMonthLength,
  getYearLength,
  getHijriToday,
} from '@misque/hijri';

// Get today's Hijri date
const today = getHijriToday();
console.log(`Today is ${today.day}/${today.month}/${today.year} AH`);

// Check if Hijri year is a leap year
isHijriLeapYear(1446); // true or false

// Get days in a Hijri month
getMonthLength(1446, 9); // 30 (Ramadan)

// Get days in a Hijri year
getYearLength(1446); // 354 or 355
```

## Date Manipulation

```typescript title="date-manipulation.ts"
import {
  addDays,
  addMonths,
  addYears,
  compareDates,
  getDaysDifference,
} from '@misque/hijri';

const date = { year: 1446, month: 9, day: 1 };

// Add days
addDays(date, 10); // { year: 1446, month: 9, day: 11 }
addDays(date, -5); // { year: 1446, month: 8, day: 26 }

// Add months
addMonths(date, 3); // { year: 1446, month: 12, day: 1 }

// Add years
addYears(date, 1); // { year: 1447, month: 9, day: 1 }

// Compare dates
const date1 = { year: 1446, month: 9, day: 1 };
const date2 = { year: 1446, month: 9, day: 15 };
compareDates(date1, date2); // -1 (date1 < date2)

// Get difference in days
getDaysDifference(date1, date2); // 14
```

## Special Date Helpers

```typescript title="special-dates.ts"
import {
  isRamadan,
  isDhulHijjah,
  getFirstDayOfMonth,
  getLastDayOfMonth,
} from '@misque/hijri';

const date = { year: 1446, month: 9, day: 15 };

// Check if in Ramadan (holy month of fasting)
isRamadan(date); // true

// Check if in Dhul Hijjah (pilgrimage month)
isDhulHijjah(date); // false

// Get first/last day of month
getFirstDayOfMonth(1446, 9); // { year: 1446, month: 9, day: 1 }
getLastDayOfMonth(1446, 9); // { year: 1446, month: 9, day: 30 }
```
