# Getting Started

This guide will help you get started with Misque libraries in your project.

## Prerequisites

- Node.js 18 or later
- A package manager (pnpm, npm, or yarn)

## Installation

## TypeScript Setup

All packages include TypeScript definitions. No additional `@types` packages needed.

```json title="tsconfig.json"
{
  "compilerOptions": {
    "moduleResolution": "bundler",
    "esModuleInterop": true
  }
}
```

## Basic Usage

### Quran

```typescript title="quran-example.ts"
import { createQuran, createQuranSession } from '@misque/quran';

// Create a Quran instance
const quran = createQuran();

// Get all surahs
const surahs = quran.getAllSurahs();

// Get a specific surah
const surah = quran.getSurah(1);
console.log(surah?.nameTransliterated); // "Al-Fatihah"

// Search the Quran
const results = quran.search('mercy');

// Work with a specific reciter
const session = createQuranSession('alzain');
const playerData = session.getPlayerData(1);
```

### Prayer Times

```typescript title="prayer-times-example.ts"
import {
  calculatePrayerTimes,
  getDefaultParams,
  formatPrayerTimes,
  getCurrentPrayer,
} from '@misque/prayer-times';

const location = {
  latitude: 51.5074,
  longitude: -0.1278,
  timezone: 'Europe/London',
};

const params = getDefaultParams();
const result = calculatePrayerTimes(new Date(), location, params);

if (result.success) {
  const times = result.data;
  const formatted = formatPrayerTimes(times);
  console.log(formatted.fajr); // "05:30"

  const current = getCurrentPrayer(times);
  console.log(current.current); // "dhuhr"
  console.log(current.next);    // "asr"
}
```

### Hijri Calendar

```typescript title="hijri-example.ts"
import { toHijri, toGregorian, formatHijri } from '@misque/hijri';

// Convert Gregorian to Hijri
const result = toHijri(new Date());
if (result.success) {
  console.log(result.data); // { year: 1446, month: 6, day: 15 }

  // Format Hijri date
  const formatted = formatHijri(result.data);
  console.log(formatted); // "15 Jumada al-Thani 1446"

  // Convert back to Gregorian
  const gregorianResult = toGregorian(result.data);
  if (gregorianResult.success) {
    console.log(gregorianResult.data); // Date object
  }
}
```

### Qibla Direction

```typescript title="qibla-example.ts"
import { calculateQibla, bearingToCompassDirection } from '@misque/qibla';

const location = { latitude: 51.5074, longitude: -0.1278 }; // London

const result = calculateQibla(location);
if (result.success) {
  console.log(result.data.bearing);          // 118.98 (degrees from North)
  console.log(result.data.distance);         // 4585.2 (km)
  console.log(result.data.compassDirection); // "ESE"

  // Or convert any bearing manually
  const compass = bearingToCompassDirection(118.98);
  console.log(compass); // "ESE"
}
```

### Mosques Finder

```typescript title="mosques-finder-example.ts"
import { findMosques, findNearestMosque } from '@misque/mosques-finder';

const location = { latitude: 51.5074, longitude: -0.1278 }; // London

// Find nearby mosques (uses OpenStreetMap data)
const result = await findMosques(location, {
  radius: 5000, // 5km radius
  limit: 10
});

if (result.success) {
  console.log(result.data.mosques); // Array of mosques with name, address, distance
  console.log(result.data.total);   // Total count
}

// Find the single nearest mosque
const nearest = await findNearestMosque(location);
if (nearest.success) {
  console.log(nearest.data.name);     // "East London Mosque"
  console.log(nearest.data.distance); // 1.2 (km)
}
```

### Zakat Calculator

```typescript title="zakat-example.ts"
import { calculateZakat, getNisabThresholds } from '@misque/zakat';

// Get current nisab thresholds (fetches live gold/silver prices)
const nisab = await getNisabThresholds('USD');
if (nisab.success) {
  console.log(nisab.data.gold);   // Nisab in USD based on 85g gold
  console.log(nisab.data.silver); // Nisab in USD based on 595g silver
}

// Calculate zakat on assets
const result = await calculateZakat({
  assets: [
    { type: 'cash', amount: 50000, currency: 'USD' },
    { type: 'gold', weight: 100, purity: 0.916 }, // 22k gold in grams
  ],
  madhab: 'hanafi', // 'hanafi' | 'shafii' | 'maliki' | 'hanbali'
  currency: 'USD',
});

if (result.success) {
  console.log(result.data.totalZakat);     // Total zakat due
  console.log(result.data.breakdown);      // Per-asset breakdown
  console.log(result.data.meetsNisab);     // Whether nisab threshold is met
}
```

## Error Handling

All Misque packages use a `Result<T>` type for error handling. This provides type-safe error handling without exceptions:

```typescript title="error-handling.ts"
import { calculateQibla } from '@misque/qibla';

const result = calculateQibla({ latitude: 999, longitude: 999 }); // Invalid coordinates

if (result.success) {
  // TypeScript knows result.data is available
  console.log(result.data.bearing);
} else {
  // TypeScript knows result.error is available
  console.error(result.error.message);
}
```
