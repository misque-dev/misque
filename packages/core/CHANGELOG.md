## 0.2.0

### Minor Changes

- Initial release of Misque - Muslim/Islamic utility libraries for JavaScript/TypeScript.

  ## Packages

  ### @misque/core

  Core utilities including Result type, coordinates, math helpers, and Julian day conversion.

  ### @misque/qibla

  Qibla direction calculation with compass support and distance to Kaaba.

  ### @misque/prayer-times

  Prayer time calculations with 14+ calculation methods (MWL, ISNA, Egypt, Makkah, etc.), high latitude support, and Asr calculation variants.

  ### @misque/hijri

  Hijri (Islamic) calendar conversion with date arithmetic and formatting.

  ### @misque/quran

  Quran data access with audio support, reciter management, and session handling.
